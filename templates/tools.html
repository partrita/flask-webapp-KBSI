Isotope Pattern Calculator

Just type in the chemical formula and this program will calculate the intensity of each mass isotope in the species. The analysis can be done in both high and low resolution. The program can also plot either calculated or manually inputted isotopic distributions, and the mass range can be scaled for better view the data. This program can calculate the isotopic distribution of a given chemical species.
