Isotope Pattern Calculator

Options


Resolution value

Number of points to use for the m/z axis (affects resolution). Default is 500. Higher is slower

Sigma value

Gaussian broadening factor (affects resolution). Default is 0.15. Lower value gives higher resolution.

About

This program can calculate the isotopic distribution of a given chemical species. This program is still under development. If you have any comments or suggestions I would like to hear from you.

Example

Key in the chemical formula for which you would like to determine the isotopic distributions. Calculation should take upwards of a few seconds depending on the size of the chemical species. The species is assumed neutral.
