Isotope Pattern Calculator

Parameter Info

Resolution

Number of points to use for the m/z axis. Default is 500. Higher values provide smoother curves but take longer to calculate.

Sigma

Gaussian broadening factor. Default is 0.15. Lower values give higher resolution (sharper peaks).

About

This program calculates the isotopic distribution of a given chemical species. The species is assumed to be neutral by default. Simply enter the formula and click Calculate.

Example

Try formulas like C6H12O6, Fe(H2O)6, or complex organic molecules. Calculation time depends on the molecular weight and complexity.
